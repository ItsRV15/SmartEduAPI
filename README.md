Smart Campus REST API

 Overview

This project is a RESTful API developed using JAX-RS (Jersey) for managing a Smart Campus environment. The system enables the management of Rooms, Sensors, and Sensor Readings through a structured and scalable API.

The API follows core REST principles including:

Resource-based architecture
Proper use of HTTP methods (GET, POST, DELETE)
Meaningful HTTP status codes
Nested resource design (sub-resources)
Structured JSON responses
Centralized error handling and logging

The system is designed to simulate a real-world campus infrastructure where sensors continuously monitor environmental conditions.
